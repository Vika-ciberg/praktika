# RentCar - Система оренди автомобілів

Веб-застосунок для оренди автомобілів, який дозволяє переглядати каталог, бронювати авто, керувати оголошеннями та читати новини водіїв.

## Технології
* **HTML5 / CSS3** (Bootstrap 5 для стилізації)
* **JavaScript (ES6+)**
* **Axios** (для HTTP-запитів)
* **LocalStorage** (для зберігання даних на клієнті)
* **JSONPlaceholder API** & **NBU API** (зовнішні дані)

## Функціонал
1.  **Авторизація:** Реєстрація та вхід (Admin/User). Збереження сесії.
2.  **Каталог:** Перегляд авто з пагінацією, пошук та фільтрація.
3.  **CRUD:** Додавання, редагування та видалення оголошень (для власників).
4.  **Бронювання:** Можливість орендувати авто та скасувати бронь.
5.  **API:** Блог новин та актуальний курс валют.
